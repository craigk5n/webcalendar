# Legacy Documentation Archive

These files are the pre-v2.0.0 HTML documentation and Perl-based doc generation toolchain.
They are preserved for historical reference only.

The current documentation is in the parent `docs/` directory in Markdown format.

## Contents

- `WebCalendar-SysAdmin.html` — System administrator guide (HTML)
- `WebCalendar-UserManual.html` — End-user manual (HTML)
- `WebCalendar-DeveloperGuide.html` — Developer guide (HTML)
- `WebCalendar-Functions.html` — Auto-generated function reference (HTML)
- `WebCalendar-Database.html` — Auto-generated database schema (HTML)
- `WebCalendar-Styling.html` — CSS styling guide (HTML)
- `faqs.html` — Generated FAQ fragment
- `sql2html.pl` — Perl script to generate DB docs from SQL comments
- `php2html.pl` — Perl script to generate function docs from PHP comments
- `extractfaqs.pl` — Perl script to extract FAQs from HTML docs
- `sql2md.py` — Python script to generate DB docs in Markdown
- `Makefile` — Build system for the Perl tools
- `event_status.gif`, `newwin.gif` — Legacy images
