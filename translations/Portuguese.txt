# Help text.  The left side is typically abbreviates since this can be long
brief-description-help: Breve descrição (cerca de 20 caracteres) do evento. Esta descrição representa o evento na visualização global do calendário
full-description-help: Descrição completa do evento. Esta informação é apresentada quando o utilizador verifica o evento isoladamente.
date-help: Especifica a data do evento.
time-help: Especifica a hora do evento<BR><I>Este campo é opcional.</I>
duration-help: Especifica a duração (em minutos) do evento.  <BR><I>Este campo é opcional.</I>
priority-help: Especifica a prioridade do evento.  Prioridade Alta é apresentada num tipo de letra mais escura.
access-help: Especifica o nível de acesso do evento.<BR> <I>Publico</I> Todos os intervenientes podem ver todos os detalhes do evento. <BR><I>Confidencial</I> Outros utilizadores poderão ver que tem um evento marcado, mas não podem visualizar os detalhes. participants-help: Lists the participants of this entry.
repeat-type-help: Selecciona a periodicidade do evento<I>Mensal (por dia)</I> faz com que o evento se repita no 1º dia do mês, 3º dia do mês, etc. <I>Mensal (por data)</I> faz com que um evento se repita todos os meses no mesmo dia.
repeat-end-date-help: Especifica a data até à qual o evento se deve repetir.
repeat-day-help: Especifica em que dias da semana o evento se deve repetir.  Isto é apenas utilizado quando o <I>Tipo de repetição</I> tem o valor de <I>Semanal</I>.
repeat-frequency-help: Especifica quantas vezes o evento se deve repetir.  O valor padrão (1) indica que o evento acontece todos os dias.  Especificar 2 faz com que o evento se repita em outra semana (se <I>Tipo de repetição</I> tem o valor de <I>Semanal</I>), todos os meses (se <I>Tipo de repetição</I> tem o valor de <I>Mensal</I>), etc.
Preferences: Configurações
Settings: Parametros
Language: Linguagem
language-help: Especifica a língua a ser utilizada.
Preferred view: Visualização preferida
preferred-view-help: Especifica a visualização padrão (Dia, Semana, Mês, ou Ano).
Time format: Formato da Hora
time-format-help: Especifica o formato da hora a utilizar: <BR><I>12 horas:</I> Mostra no formato 3am, 8:30pm, etc.  <BR><I>24 horas:</I> Mostra no formato 300, 2030, etc.
Display unapproved: Visualizar Não Aprovados
display-unapproved-help: Especifica se eventos não aprovados são mostrados no calendário.
Display icons: Visualizar ícones
display-icons-help: Mostra ícones junto de cada evento para visualizar, editar e apagar.
Display week number: Visualizar numero da semana
display-week-number-help: Especifica se o número da semana (1-52) deve ser mostrado na vista semanal e mensal.
display-week-starts-on: Especifica se a semana começa ao Domingo ou à Segunda-Feira. 
Work hours: Horário de trabalho
work-hours-help: Especifica as horas apresentadas para marcação de eventos.
Colors: Cores
colors-help: Todas as cores devem ser especificadas da forma "#RRGGBB" hexadecimal onde "RR" é o valor hexadecimal para o encarnado, "GG" é o valor hexadecimal para o verde, e "BB" o valor hexadecimal para o azul.
